# Orfograffiti

**Orfograffiti** — это образовательная игра на платформе .NET MAUI, которая помогает улучшить навыки правописания и расширить словарный запас через игровую механику угадывания слов.

## 📱 Платформы

- Android
- iOS
- macOS (Mac Catalyst)
- Windows

## 🎮 Игровые режимы

### Классическая игра
- 5 слов за раунд
- Три уровня сложности: Лёгкий, Нормальный, Сложный
- Открытие случайных букв в начале раунда
- Ограниченное количество попыток

### ГиперИгра
- 10 слов за раунд
- Повышенная награда за победу
- Требует больше концентрации и словарного запаса

### Гонка
- Игра на время с ограниченными попытками
- 8/7/6.5 минут в зависимости от сложности
- Специальное достижение за победу на последних секундах

## 🏆 Система достижений

| Достижение | Условие | Награда |
|------------|---------|---------|
| Новичок | Выиграть 1 игру | ✓ |
| Продвинутый | Выиграть 10 игр | ✓ |
| Мастер слов | Выиграть 100 игр | ✓ |
| Поражение | Проиграть 1 игру | ✓ |
| Невежда | Проиграть 10 игр | ✓ |
| Нубяра | Проиграть 100 игр | ✓ |
| Вовремя! | Победа при таймере < 5 сек | ✓ |
| Фижма | Угадать секретное слово | ✓ |

## 📊 Ранговая система

Опыт накапливается за победы в играх:

| Ранг | Требуемый опыт |
|------|----------------|
| Деревянный | 0 — 5 000 |
| Железный | 5 000 — 12 000 |
| Бронзовый | 12 000 — 25 000 |
| Серебряный | 25 000 — 40 000 |
| Золотой | 40 000 — 60 000 |
| Изумрудный | 60 000 — 90 000 |
| Рубиновый | 90 000 — 130 000 |
| Неоновый | 130 000 — 200 000 |
| Престиж | 200 000+ |

При достижении нового ранга показывается анимированное уведомление.

## 🎯 Игровая механика

1. В начале раунда открываются случайные буквы (количество зависит от сложности)
2. Игрок вводит предполагаемые слова в пустые ячейки
3. После проверки угаданные буквы открываются во всех словах
4. Игра продолжается пока не будут угаданы все слова или не закончатся попытки
5. За каждое угаданное слово игрок получает опыт и букоины

### Сложности

| Сложность | Попытки | Макс. длина слова | Открытых букв |
|-----------|---------|-------------------|---------------|
| Лёгкая | 12 | 5 | 4 |
| Нормальная | 9 | 7 | 3 |
| Сложная | 7 | 8 | 2 |

## 🔧 Технические особенности

- **Фреймворк**: .NET MAUI
- **Язык**: C# + XAML
- **Библиотеки**: CommunityToolkit.Maui
- **Хранение данных**: Preferences (локальное хранилище)
- **Поддержка темной/светлой темы**: динамическая смена через ThemeApp

## 🚀 Установка и запуск

### Требования
- .NET 9 SDK
- Visual Studio 2022 (или VS Code с расширениями MAUI)
- Для Android: Android SDK API 21+
- Для iOS: Xcode 15+ (macOS)
- Для Windows: Windows 10 версии 19041+

### Клонирование и сборка

git clone https://github.com/yourusername/Orfograffiti.git
cd Orfograffiti
dotnet restore
dotnet build


### Запуск на конкретной платформе

# Windows
dotnet build -f net9.0-windows10.0.19041.0

# Android
dotnet build -f net9.0-android

# iOS (требуется macOS)
dotnet build -f net9.0-ios

## 📝 Файлы словарей

Игра использует текстовые файлы со словарями:
- `words.txt` — основной словарь для проверки существования слов
- `easywords.txt`, `normalwords.txt`, `hardwords.txt` — словари для генерации заданий

## 🎨 Кастомизация

### Добавление новой темы
1. Добавьте запись в `ThemeApp.ThemeDict`
2. Добавьте соответствующий фоновый рисунок в `Resources/Images/`
3. Добавьте тему в магазин в `shop.xaml.cs`

### Добавление нового шрифта
1. Поместите файл шрифта в `Resources/Fonts/`
2. Добавьте его в `MauiProgram.cs` через `ConfigureFonts`
3. Добавьте шрифт в магазин в `shop.xaml.cs`
